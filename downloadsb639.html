Looking for products? Try Caedium Product Downloads.

Downloads are grouped by file type as follows:

  • sym (.sym) – native Caedium file, can be viewed with Caedium
  • BRep (.brep) – geometry file, can be viewed with Caedium using File->Import
  • STEP (.stp) – geometry file, requires Caedium Exchange add-on, then can be viewed with Caedium using File->Import
  • IGES (.igs) – geometry file, requires Caedium Exchange add-on, then can be viewed with Caedium using File->Import
  • Alias/Wavefront (.obj) – faceted geometry file, requires Caedium Exchange add-on, then can be viewed with Caedium using File->Import
