No rotating mesh

Submitted by symscape on December 27, 2011 - 22:10.

Currently Caedium can not simulate a rotating turbine, which requires either Moving Reference Frame (MRF) or moving mesh. However, Caedium should be able to mesh the geometry. For help on diagnosing meshing problems see "How do I fix mesh and solver failures?".

MRF now supported in Caedium

Submitted by symscape on May 25, 2013 - 15:27.

MRF is now supported in Caedium. For more details see the tutorial "Rotating Paddle using Moving Reference Frame".
