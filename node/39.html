Tools

Submitted by symscape on May 24, 2013 - 16:38

All of the tools in the Caedium Tool Palette either create or modify a selection in the View Window, where a selection can be one or more geometry entities, or the simulation.

View our screencast on the role of tools in Caedium:

Applying a Tool

There are 4 ways to add (or apply) a tool to a selection as follows:

  • Drag and Drop: Drag and drop the tool onto the entities in the View Window.
  • Copy and Paste: Right click the tool (launching the context popup menu) and select copy. Then right click on the entities in the View Window to select them, and select paste in the context popup menu.
  • Double Click: Right click on the entities in the View Window to select them. Then double click the tool to apply it. If a tool is incompatible with the current selection then double clicking it will have no effect
  • Add Button: Right click on the entities in the View Window to select them. Then click the 'Add {tool name}' button in the Properties Panel. If a tool is incompatible with the current selection then the 'Add {tool name}' button will be disabled.

Tool Properties

Most tools have associated properties that are displayed in the Properties Panel when a tool is selected. Changing a tool's properties will determine how the tool operates when it is applied to geometry entities or the simulation.

For example, to change the size of the box created by the Box tool, in the Geometry Tool Palette select the Volumes->Box tool and then modify the box dimensions in the Properties Panel using the Lengths property directly or using the individual component properties X Delta, Y Delta, and Z Delta, as shown below.

Box Tool Properties

The next time you use the Box tool it will create a box with your new dimensions.
