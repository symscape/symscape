Install libwebkitgtk manually

Submitted by symscape on August 15, 2012 - 10:05.

Try installing libwebkitgtk according to:
sudo apt-get install libwebkitgtk-1.0-0

You might also need to install libwebkitgtk-1.0-common based on the discussion at http://ubuntuforums.org/archive/index.php/t-1767948.html

Please report back as to whether this solves your problem.
