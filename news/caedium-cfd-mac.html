Caedium Brings Interactive CFD to the Mac

Submitted by symscape on March 6, 2010 - 12:28
  • News

The latest release of Caedium (v2.1) now supports Mac OS X 10.5 (Leopard) and 10.6 (Snow Leopard) in addition to Windows and Linux. Also in this new release are various enhancements to 3D selection, the properties panel, and geometry creation for all platforms.

Caedium on the Mac

Mac OS X

With this release Caedium becomes one of the few (maybe the only) commercial CFD analysis tools with a rich interactive GUI to run as a native application on the Mac. Functionally the Mac version is identical to the Windows and Linux versions of Caedium, so engineers can choose their most productive platform to perform CFD simulations.

The relative ease of producing the native Mac version along with the existing Windows and Linux versions demonstrates the unique cross-platform architecture embedded within Caedium from its initial conception. This architecture means that as enhancements and new features are added to Caedium, they will be available across all platforms simultaneously.

Enhancements

There are also several major and minor enhancements across many of the Caedium add-ons for all platforms in this latest release, such as:

  • Improved 3D selection speed and shortcut key conveniences
  • The properties panel now switches tabs and expands new properties, e.g., when dragging and dropping an inlet boundary condition onto a face
  • ASCII point sets can be imported as vertices and groups for easy spline fitting, which is ideal for defining airfoils
  • Creation of a rectangular grid as the basis for easy geometry placement and construction

For a detailed list of enhancements and bug fixes see the Caedium v2.1 release notes.

Caedium is available to download and install directly from our website. We offer free 30-day trials and affordable subscriptions for Caedium add-ons.

Caedium v2.1.1 Bug Fix Release

Submitted by symscape on March 16, 2010 - 12:44.

The latest version of Caedium (v2.1.1) contains a number of important bug fixes, mainly for the new Mac OS X version.

Bugs fixed for Caedium on Mac OS X are:

  • Various incompatibilities with OS X 10.5 (Leopard) prevented Caedium from starting without errors, displaying OpenGL View Windows, and running OpenFOAM flow simulations
  • License activation failures with restart
  • High frequency flicker on some ATI graphics cards

For the full list of bug fixes see the Caedium v2.1.1 release notes.
