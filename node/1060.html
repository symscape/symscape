Consultant Guidelines

Submitted by symscape on March 23, 2011 - 09:13
  • Consultants

Thankyou for considering our CFD Consultants forum to promote your talents.

To get started you'll need to either:

  • Login
  • Create a free Symscape account

We suggest the following profile format:

  • Name
  • Expertise
  • Experience
  • Portfolio
  • Rate

For example:

Name
CFD'R'US
Expertise
External Aerodynamics
Experience
5 years at leading F1 CFD consultancy using Caedium
Portfolio
http://www.symscape.com/
Rate
Minimum $200/hr, dependent on size and scope of project

<dd>CFD'R'US</dd>

<dd>External Aerodynamics</dd>

<dd>Minimum $200/hr, dependent on size and scope of project</dd>
</dl>

Be sure to post additional comments to your profile when you have updates (e.g., news, examples) you want to share.

Please use the personal Symscape account contact system to communicate directly with prospects rather than posting email addresses in these forums. Make sure your personal Symscape account has the Personal contact form option enabled - this option is in the Edit section in My account.

If you want to post a comment on a project proposal, first login with your Symscape account ID. Anonymous comments will be deleted.
