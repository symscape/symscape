Calculating Induced Drag in Panel Flow

Submitted by David O'Donnell on November 23, 2011 - 19:50
  • Questions

I am using Caedium Panel Flow to model a wing section for a sailplane as part of my undergraduate project. I have completed the NACA 4415 tutorial and understand how to calculate Cp, but I want to get the induced drag distribution too for my wing section too.

Is there a way to calculate this using Caedium? Also, can I import Autocad (.dwg, .dxf) files into Panel Flow to be solved?

Any assistance would be greatly appreciated.

David O'Donnell

Create Drag Force Monitor

Submitted by symscape on November 23, 2011 - 20:51.

Caedium Panel Flow can calculate the drag force (minus the skin friction) on your wing, but you can not separate the pressure drag from the induced drag. For a finite wing (rather than the pseudo infinite wing in the tutorial) you will want to apply a wall boundary condition to the wing tip.

To create a force monitor see the section "Create Force Monitors" in the tutorial "Flow Over a Rotating Wheel with Moving Ground".

Caedium can not import Autocad files. If you have the option try using STEP (.stp). If not you can try Wavefront (.obj). For more details see "Geometry Exchange".
