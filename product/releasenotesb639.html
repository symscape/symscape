Caedium Release Notes

Release notes provide descriptions of enhancements and bug fixes for each release of Caedium and its add-ons.

If an enhancement or bug fix is specific to an add-on, the add-on is listed in square brackets, e.g., [Builder]. Note that all enhancements and bug fixes are included in the Professional add-on, which is a collection of all Caedium add-ons.

Known Issues

  • If a media player does not launch after exporting an animation file (.avi), select File->Preferences, and then in the Properties Panel select the Export->Movies property, right-click on it, and select Reset.
  • Windows
    • Preview and playback of an animation file (.avi) created in Caedium requires Windows Media Player.
  • Mac OS X
    • Gatekeeper prevents Caedium from installing. For a workaround see "Caedium Workaround for OS X 10.8 Gatekeeper"
    • /bin/sh requires Full Disk Access according to Mac OS X error: shell-init: error retrieving current directory

Caedium Version 2.4.0 Release Notes

Submitted by symscape on December 2, 2010 - 11:52
  • Release Notes

The latest Caedium release contains a variety of enhancements and bug fixes. One of the highlights is the ability to produce high quality rendered movies (animations) with the option to use either POV-Ray or a Renderman-compliant application (e.g., Aqsis) to produce each movie frame. Frames can be rendered in parallel (either on standalone multi-core computers or on a cluster running Windows HPC Server 2008) in order to significantly speed up the overall movie creation process.

Another highlight of this Caedium release is the improved interactive performance when manipulating the 3D view and selecting entities. Plots and monitors now have interactive panning and zooming.

Caedium Version 2.3.0 Release Notes

Submitted by symscape on September 7, 2010 - 13:52
  • Release Notes

The OpenFOAM®1 solvers and utilities used by the Caedium RANS Flow and Professional add-ons have been upgraded to OpenFOAM 1.7.x from 1.6.x. Also the unsteady incompressible flow solvers in Caedium now use PIMPLE instead of PISO. PIMPLE has better stability for large time steps compared to PISO.

Caedium Version 2.2.0 Release Notes

Submitted by symscape on August 6, 2010 - 13:23
  • Release Notes

The Caedium RANS Flow and Professional add-ons can now perform CFD simulations in parallel, either on a single machine with multiple cores or on a Windows cluster, which can lead to a significant reduction in simulation turnaround time. Also Windows and Linux versions of Caedium are now 64-bit, which means Caedium can perform much larger simulations (no longer being limited to the 32-bit memory maximum of 2-4 GB).
